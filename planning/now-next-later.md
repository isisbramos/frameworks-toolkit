# Now / Next / Later Roadmap

## When to use
When communicating the roadmap without committing to fake dates. Great for stakeholder alignment and team focus.

## Template

### Now (Current focus)
> What are we actively working on right now?
>
> | Initiative | Goal | Status |
> |-----------|------|--------|
> | | | |
>
> ### Next (Up next)
> > What are we planning to work on after current work is done?
> >
> > | Initiative | Goal | Confidence |
> > |-----------|------|------------|
> > | | | |
> >
> > ### Later (On the horizon)
> > > What are we thinking about but haven't committed to?
> > >
> > > | Initiative | Goal | Notes |
> > > |-----------|------|-------|
> > > | | | |
> > >
> > > ## How to use it
> > >
> > > - "Now" = committed, in progress
> > > - - "Next" = high confidence, planned
> > >   - - "Later" = exploring, not committed
> > >     - - Review and update weekly or bi-weekly
> > >       - - Share with stakeholders as the primary roadmap artifact
> > >        
> > >         - ## Tips
> > >        
> > >         - - Resist the urge to put dates on "Next" and "Later"
> > >           - - If something stays in "Later" for too long, either promote it or kill it
> > >             - - Keep each bucket to 3-5 items max for clarity
